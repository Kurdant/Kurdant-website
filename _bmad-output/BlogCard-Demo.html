Notes de la demo

  • Hover sur une card pour voir le glitch effect et le glow
  • Le scale est subtil (1.03) pour éviter les layout shifts
  • Les cards sont responsives: 1 colonne mobile, 2 tablette, 3 desktop
  • Les tags ont un background semi-transparent avec bordure
  • La date utilise JetBrains Mono (monospace) en uppercase
  • L'excerpt est tronqué après 3 lignes avec ellipsis
  • Les images utilisent object-fit: cover pour rester proportionnées
  • Focus state visible pour accessibilité keyboard (outline cyan)
  • Respecte prefers-reduced-motion pour utilisateurs sensibles
